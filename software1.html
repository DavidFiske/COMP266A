Rock, Paper, Scissors, Lizard, Spock

Version 1.1
Language: Java

This is a basic 1 player version of the rock, paper, scissors, lizard, spock game that I wrote for my intro to Java class.

When the program runs, you will automatically select rock, paper, scissors, lizard, or spock and the computer will do the same. In the event of a tie, you will both automatically select one of the five options again.
